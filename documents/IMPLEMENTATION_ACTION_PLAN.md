## PHASE 2: SEO ENHANCEMENT (Week 2-3 - 2.5 hours)
**Impact:** 15-25% CTR improvement, featured snippet opportunities, better rankings

### 1. Add JSON-LD Schema Markup
**Location:** Components needed in multiple pages
**Priority:** HIGH (Easy to implement, high ROI)

#### Implementation Items:

**A. LocalBusiness Schema** (Homepage)
\`\`\`typescript
// Add to /app/page.tsx
const localBusinessSchema = {
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "PowerWash Bros",
  "description": "Professional pressure washing and exterior cleaning in Swanage, Purbeck & Dorset",
  "url": "https://powerwashbros.co.uk",
  "telephone": "+447418610731",
  "email": "info@powerwashbros.co.uk",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "Swanage",
    "addressRegion": "Dorset",
    "postalCode": "BH19",
    "addressCountry": "GB"
  },
  "priceRange": "$$",
  "areaServed": [
    "Swanage",
    "Purbeck",
    "Dorset"
  ],
  "sameAs": [
    "https://www.facebook.com/powerwashbros",
    "https://www.instagram.com/powerwashbros"
  ]
}
\`\`\`

**B. Service Schema** (Service pages)
\`\`\`typescript
// Add to /app/services/[service]/page.tsx
const serviceSchema = {
  "@context": "https://schema.org",
  "@type": "Service",
  "name": "Gutter Cleaning",
  "provider": {
    "@type": "LocalBusiness",
    "name": "PowerWash Bros"
  },
  "areaServed": "Dorset, UK",
  "serviceType": "Exterior Cleaning"
}
\`\`\`

**C. FAQPage Schema** (FAQ page)
\`\`\`typescript
// Add to /app/faq/page.tsx
const faqSchema = {
  "@context": "https://schema.org",
  "@type": "FAQPage",

    {
      "@type": "Question",

      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Pressure washing uses high-pressure water to clean surfaces..."
      }
    }
    // More FAQ items
  ]
}
\`\`\`

**D. BreadcrumbList Schema** (All pages)
\`\`\`typescript
